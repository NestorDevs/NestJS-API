## Description

[Nest](https://github.com/nestjs/nest)JS API (PostgreSQL/TypeORM) playground. In progress.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development & watch mode
$ npm run dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
