## Installation

```bash
$ npm install
```

## Running the app

```bash
# docker
$ docker-compose up -d

# development & watch mode
$ npm run dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
